## Deskripsi
Aplikasi kalkulator cerdas untuk studi kelayakan bisnis pabrik keripik buah. Membantu menghitung BEP, ROI, dan Profit Margin secara instan tanpa perlu login.

## Rumus yang Digunakan

### 1. Break Even Point (BEP)
Menentukan titik impas dimana pendapatan sama dengan pengeluaran.
- **BEP (Unit)** = Biaya Tetap / (Harga Jual - Biaya Variabel per Unit)
- **BEP (Rupiah)** = BEP Unit × Harga Jual

### 2. Return on Investment (ROI)
Estimasi berapa bulan modal akan kembali.
- **ROI (Bulan)** = Total Investasi Awal (CAPEX) / Laba Bersih per Bulan

### 3. Laba Bersih
- **Laba Kotor** = (Harga Jual - HPP) × Kapasitas Produksi
- **Laba Bersih** = Laba Kotor - Biaya Tetap Operasional
